# OzymandiasUTN

Trabajo Práctico Integrador de Programación I (UTN FRGP). Juego de consola para dos jugadores inspirado en Breaking Bad: Walter White y Gale Boetticher compiten durante 6 rondas cocinando cristal con dados, administrando Metilamina y evitando a la DEA. Gana quien acumule $737.000 primero, o el que tenga más dinero al final.
